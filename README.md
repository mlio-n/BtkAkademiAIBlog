# BTK Akademi - AI Blog Web API

Bu proje, BTK Akademi platformunda yer alan "ASP.NET Core ile API Tabanlı Yapay Zeka Atölyesi" eğitimi kapsamında, eğitim süreciyle eş zamanlı olarak geliştirilmektedir. Projenin temel amacı, eğitimde öğrenilen kurumsal backend mimarilerini ve yapay zeka entegrasyonlarını anında pratiğe dökerek canlı bir Web API altyapısı inşa etmektir.

## Kullanılan Teknolojiler (Tech Stack)
* Backend: C#, ASP.NET Core 10.0
* API Dokümantasyonu: Swagger / OpenAPI
